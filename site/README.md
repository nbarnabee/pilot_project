# Fruitcake ProcessWire Template

This template is used as a default starting point for projects using ProcessWire.

It contains a basic version of the following tools/functions:

* Basic responsive CSS container and grid framework (incl. modern CSS reset)
* Contact Form template with Google reCaptcha (disabled by default)
* Support for ProcessWire's SeoMaestro module
* Support for basic multilingual sites (hreflang meta)
* Tiny Slider 2 JS library

## Installation and usage

Refer to the documentation [here](https://github.com/fruitcakeweb/fruitcake-documentation/blob/main/processwire/project-setup.md).

For local development installations, you should rename the `config.php` to `config-dev.php`.
